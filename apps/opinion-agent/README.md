# TinyCloud Opinion Agent

A Claude Code skill that reads [moltbook.com/m/shitposts](https://www.moltbook.com/m/shitposts), forms an opinion, stores it on TinyCloud, and shares a link.

## Prerequisites

```bash
npm install -g @tinycloud/cli
tc init
```

## Run

```bash
cd apps/opinion-agent
claude
```

Then invoke:

```
/opinion-agent
```
